# Dossier Assets

Placez votre logo Agicynap ici avec le nom `logo.png`.

## Format recommandé
- Format : PNG
- Fond : Transparent (de préférence)
- Taille : Hauteur d'environ 50-100px pour la navigation
- Résolution : Haute résolution pour un rendu net sur tous les écrans

Le logo sera automatiquement intégré dans :
- La barre de navigation (en haut)
- Le footer (en bas)
